Simple React application demonstrating props, state management, and event handling for exam purposes.

## How to Run (5 Steps)

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Start Development Server
```bash
npm run dev
```

### Step 3: Open in Browser
- The app will open automatically, or
- Manually open: `http://localhost:3000` (or the port shown in terminal)

### Step 4: Navigate Between Demos
- Click **"Props & State Demo"** to see the first assignment
- Click **"Event Handling Demo"** to see the event handling assignment
- Click **"Home"** to return to the main menu

### Step 5: Test the Features

**Props & State Demo:**
1. Shows initial message: "Hello, User! You are 20 years old"
2. Click "Update Info" button
3. Name changes to "Aswini" and age increments

**Event Handling Demo:**
1. Click "Click Me" button → Shows "Button clicked"
2. Hover over "Hover Me" button → Shows "Mouse Hovered"
3. Type in input field → Shows "Typed: [your text]"

## What's Included

- **App.jsx** - Props & State demonstration
- **EventDemo.jsx** - Event handling (onClick, onMouseOver, onChange)
- **MainApp.jsx** - Simple navigation between demos
